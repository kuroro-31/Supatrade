## 環境
- フロント： 
React, Next.js, Typescript
- Paas: Vercel
- CICD: Vercel
- DB：　Supabase
- CDN： Cloudflare
- 状態管理：　Zostand
- スタイル： TailwindCSS, Sass, Postcss
- フォント: Noto Sans JP
- コード分析: Eslint
- コード整形： Prettier
- 分析: vercel analytics
- 単体テスト: Jest
